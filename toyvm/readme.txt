ВНИМАНИЕ!!!

Исходные *.java коды были подготовлены в IDE NetBeans 8.2.
Если Вы будете компилировать исходные коды в операционной системе MS Windows не в IDE,
а из командной строки, то может оказаться необходимым указать компилятору дополнительную
опцию кодировки.

вариант 1: javac -encoding Cp1251 toystackmachine/*.java
вариант 2: javac -encoding UTF-8 toystackmachine/*.java


Более подробно можно почитать тут: http://www.skipy.ru/technics/encodings_console_comp.html
Исторически, к сожалению, в MS Windows с этим давняя неразбериха.
